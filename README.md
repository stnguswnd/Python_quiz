# Git과 함께하는 Python 첫 발자국 - 콘솔 퀴즈 게임

## 1. 프로젝트 개요
이 프로젝트는 Python으로 만든 콘솔 기반 퀴즈 게임입니다.  
사용자는 메뉴를 통해 퀴즈를 풀고, 새로운 퀴즈를 추가하고, 등록된 퀴즈 목록을 확인하고, 최고 점수를 조회할 수 있습니다.  
또한 프로그램을 종료한 뒤 다시 실행해도 퀴즈 데이터와 최고 점수가 유지되도록 `state.json` 파일을 활용해 저장 기능을 구현했습니다.

이 프로젝트를 통해 다음 내용을 함께 연습할 수 있었습니다.

- Python 기본 문법
- 함수 분리와 프로그램 구조화
- 클래스 설계 (`Quiz`, `QuizGame`)
- 파일 입출력 및 JSON 데이터 저장
- 예외 처리
- Git을 활용한 버전 관리와 브랜치 작업

---

## 2. 퀴즈 주제 선정 이유
제가 선택한 퀴즈 주제는 **영화 퀴즈**입니다.

영화는 평소에도 자주 접하는 분야라 문제를 직접 만드는 과정이 재미있었고, 많은 사람이 익숙하게 접근할 수 있는 주제라고 생각했습니다.  
또한 감독, 배우, 줄거리, 시리즈 작품 등 다양한 기준으로 문제를 만들 수 있어 최소 5개 이상의 문제를 구성하기에도 적절했습니다.  
정답이 너무 어렵지 않으면서도 적당한 재미를 줄 수 있다는 점에서 콘솔 퀴즈 게임의 주제로 잘 맞는다고 판단했습니다.

---

## 3. 주요 기능

### 1) 퀴즈 풀기
- 저장된 퀴즈를 순서대로 출력합니다.
- 사용자는 각 문제의 정답 번호를 입력할 수 있습니다.
- 정답/오답 여부를 즉시 확인할 수 있습니다.
- 모든 문제를 다 풀면 최종 점수를 출력합니다.
- 기존 최고 점수보다 높으면 최고 점수를 갱신합니다.

### 2) 퀴즈 추가
- 사용자가 새로운 퀴즈를 직접 등록할 수 있습니다.
- 문제, 선택지 4개, 정답 번호를 입력받습니다.
- 잘못된 입력이 들어오면 다시 입력받도록 처리했습니다.
- 추가된 퀴즈는 프로그램 재실행 후에도 유지됩니다.

### 3) 퀴즈 목록 확인
- 현재 저장된 전체 퀴즈 목록을 확인할 수 있습니다.
- 문제 번호와 문제 내용을 출력합니다.

### 4) 점수 확인
- 현재 저장된 최고 점수를 확인할 수 있습니다.

### 5) 종료
- 프로그램을 안전하게 종료합니다.
- 종료 전 필요한 데이터가 파일에 저장되도록 구성했습니다.

## 6. 파일 구조
```text
project/
├─ main.py
├─ quiz.py
├─ quiz_game.py
├─ state.json
├─ README.md
└─ docs/
   └─ screenshots/
      ├─ menu.png
      ├─ play.png
      ├─ add_quiz.png
      └─ score.png
```

### 파일 설명
- `main.py`  
  프로그램 시작 파일입니다. 전체 실행 흐름을 담당합니다.

- `quiz.py`  
  퀴즈 1개를 표현하는 `Quiz` 클래스를 정의한 파일입니다.

- `quiz_game.py`  
  게임 전체 로직을 담당하는 `QuizGame` 클래스를 정의한 파일입니다. 메뉴 출력, 퀴즈 진행, 퀴즈 추가, 목록 확인, 점수 확인, 파일 저장/불러오기 기능이 포함됩니다.

- `state.json`  
  퀴즈 목록과 최고 점수를 저장하는 데이터 파일입니다.

- `README.md`  
  프로젝트 설명 문서입니다.

- `docs/screenshots/`  
  실행 화면 캡처 이미지를 저장하는 폴더입니다.

## 7. 클래스 설계

### Quiz 클래스
개별 퀴즈 1개를 표현하는 클래스입니다.

#### 속성
- `question` : 문제 내용
- `choices` : 선택지 목록
- `answer` : 정답 번호

#### 역할
- 퀴즈 1개의 데이터를 저장합니다.
- 문제와 보기, 정답 정보를 관리합니다.

---

### QuizGame 클래스
게임 전체를 관리하는 클래스입니다.

#### 속성
- `quizzes` : 전체 퀴즈 목록
- `best_score` : 최고 점수

#### 역할
- 메뉴 출력
- 퀴즈 풀기
- 퀴즈 추가
- 퀴즈 목록 출력
- 최고 점수 확인
- 파일 저장 및 불러오기

## 9. 데이터 파일 설명 (`state.json`)
이 프로젝트는 프로그램 종료 후에도 데이터가 유지되도록 `state.json` 파일을 사용합니다.  
이 파일에는 퀴즈 목록과 최고 점수가 저장됩니다.

### 저장 예시
```json
{
  "quizzes": [
    {
      "question": "영화 '기생충'의 감독은 누구인가?",
      "choices": ["박찬욱", "봉준호", "김기덕", "이창동"],
      "answer": 2
    },
    {
      "question": "마블 시네마틱 유니버스에서 타노스가 모은 인피니티 스톤의 개수는?",
      "choices": ["4개", "5개", "6개", "7개"],
      "answer": 3
    }
  ],
  "best_score": 80
}
```

### 각 필드 설명
- `quizzes`  
  저장된 전체 퀴즈 목록입니다.

- `question`  
  문제 문자열입니다.

- `choices`  
  보기 4개를 담는 리스트입니다.

- `answer`  
  정답 번호입니다.  
  예: 1 ~ 4 중 하나

- `best_score`  
  현재까지 기록된 최고 점수입니다.

## 10. 예외 처리 및 입력 검증
프로그램이 실행 중 갑자기 종료되지 않도록 입력값 검증과 예외 처리를 적용했습니다.

처리한 예시는 다음과 같습니다.

- 숫자가 아닌 값 입력
- 메뉴 범위를 벗어난 값 입력
- 퀴즈 정답 입력 시 잘못된 번호 입력
- 빈 입력
- `state.json` 파일이 없는 경우
- `state.json` 파일이 손상된 경우
- 프로그램 실행 중 `Ctrl + C` 또는 입력 종료가 발생한 경우
